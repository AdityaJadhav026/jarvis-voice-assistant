# Jarvis Voice Assistant (Python)

This is a Python-based voice assistant project using speech recognition and text-to-speech.

## Features
- Voice activation using keyword "Jarvis"
- Open websites like Google, YouTube, Facebook, LinkedIn
- Speech-to-text and text-to-speech

## Requirements
- Python 3.x
- Microphone
-SpeechRecognition
-pyttsx3
-pyaudio


## Install Dependencies
pip install -r requirements.txt

## Run Project
python main.py
